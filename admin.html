Геометрия сцены
Загрузите фото и выставьте базовый контур сцены.
Сначала подготовьте базовую геометрию сцены: фото → контур → вырезы → просмотр. После этого захватывайте сцену в local draft и переходите к ручной настройке вариантов текстур.
Редактор варианта (local draft)
Выберите форму и текстуру, затем захватите текущий вариант.
Вариант сохраняет ручные настройки формы, текстуры, параметров материала, camera/AI state и активной зоны поверх уже подготовленной базовой сцены. Это foundation для последующего published workflow.
Local draft сохраняется в браузере и подходит для authoring без backend. Для надёжного backup или переноса используйте экспорт scene.json.
Repo structure autofill
Настройте base URL и шаблоны имён файлов, затем автозаполните URL сцены и preview URL вариантов.
Helper использует единый шаблон GitHub Pages URL и строит publish-ready URL для photo/thumb/cover и preview каждого local variant. Это снижает ручную рутину перед экспортом пакета сцены.
Bulk asset import helper
Вставьте список URL или путей ассетов одной пачкой. Helper попробует автоматически сопоставить scene photo/thumb/cover и preview по вариантам.

3D‑калибровка перспективы (Beta)
Для профессиональной перспективы отметьте 4 линии: A1+A2 — вдоль направления укладки (в глубину), B1+B2 — поперёк. После калибровки параметры «Перспектива/Горизонт» будут выставлены автоматически с сохранением правила: укладка всегда идёт снизу вверх.
